# Personal Portfolio Website


[markslorach.com](https://www.markslorach.com/) is my personal portfolio website built using [Next.js](https://nextjs.org/), [Tailwind](https://tailwindcss.com/), [Framer Motion](https://www.framer.com/motion/) and [Sanity.io](https://www.sanity.io/) as a Content Management System.

### Quick Start

Install the packages in the root folder.
```bash
npm install
```

To run the project.
```bash
npm run dev
```

This project is deployed through [Vercel](https://vercel.com/).
